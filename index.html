📚 Como Usar

Guia rápido para começar a usar o Minecraft Server Manager

1

📁 Preparação

Coloque o executável na pasta do seu servidor Minecraft Bedrock, junto com bedrock_server.exe e server.properties

2

🚀 Execução

Execute o MinecraftServerManager.exe. O programa detectará automaticamente os arquivos do servidor

3

🎮 Gerenciamento

Use as abas para controlar o servidor, configurar propriedades, gerenciar mundos, addons e jogadores

4

🌍 Playit (Opcional)

Para acesso pela internet, baixe o Playit.gg e coloque na pasta playit/ dentro do servidor

📁 Estrutura Recomendada

servidor_minecraft/
├── MinecraftServerManager.exe  ← Este arquivo
├── bedrock_server.exe
├── server.properties
├── worlds/
├── development_behavior_packs/
├── development_resource_packs/
└── playit/
    └── playit.exe  ← Para acesso pela internet
